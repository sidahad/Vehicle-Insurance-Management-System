# Vehicle Insurance Management System

An ASP.NET Core MVC application for managing vehicle insurance policies, claims, customer records, and premium payments.

## Features

* User Authentication & Authorization
* Vehicle Insurance Policy Management
* Insurance Claims Processing
* Premium Payment Tracking
* Customer Management
* Admin Dashboard
* Policy Renewal Management
* Responsive UI

## Technologies

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* HTML5
* CSS3
* JavaScript
* Bootstrap
